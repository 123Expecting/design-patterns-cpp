# design-patterns
《大话设计模式》中23种设计模式案例的C++版本实现

## 组织结构
* src - 每个模式案例的声明（.h）和实现（.cc）
* tests - 每个模式案例的gtest，相当于客户端
* docs - 每个模式案例的UML（.html）

## 编译结果
* patterns - src编译得到的模式案例类库
* patterns_run - main输出设计模式字符图
* patterns_test - tests中所有案例的单元测试
